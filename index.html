Essential image fader sample demo

This is a sample demonstration of my code. Feel free to modify for your personal needs!

In order to use this code in your website, the following files are needed:

  • The jQuery javascript plugin (I used version 2.1.3, you can find it on Google here).
  • The javscript file imfade-0.2.js (you can find it in the "js" folder in the script)
  • The css file imfade.css needed for the correct styling of the html elements

To make the code work properly, just follow this simple steps:

1) Add links to the javascript and css files inside the head of your html document. Create also a link on the jQuery plugin that is needed in order to run the imfader-0.2.js javascript file. I used the version 2.1.3.


<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
<script src="js/imfade-0.1.js"></script>
<link href="imfade.css" rel="stylesheet" type="text/css" />

2) Create a <div> element inside the body of your html document where you want to put your slideshow. Here is an example on how to properly set-up the slideshow:


<div id="imfade-container">
    <div id="imfade-images">
        <img class="active" src="images/im01.jpg" title="This is a caption to the first image.">
        <img src="images/im02.ipg" title="This is a caption to the second image.">
        <img src="images/im03.jpg">
    </div>
</div>

3) Run the fader by calling the imageFader method of javascript plugin jQuery. The method should be called on the HTML <div> that contains the images, here in this example it's the one having ID imfade-container.


<script>
	jQuery(document).ready(function($) {
		jQuery('#imfade-container').imageFader({
			frameWidth 	: '100%',
			frameHeight : '480px'
		});
	});
</script>

The method can be called with some options as shown above. For the full list of options see the source file imfader-0.2.js.
